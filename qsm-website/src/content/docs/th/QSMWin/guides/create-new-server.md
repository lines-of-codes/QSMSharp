title: การสร้างเซิร์ฟเวอร์ใหม่
description: การสร้างเซิร์ฟเวอร์ Minecraft ใหม่ด้วย QSM
---

หลังจากทำการติดตั้ง QSM คุณสามารถสร้างเซิร์ฟเวอร์ใหม่ได้โดยการไปยังแท็บ "สร้างเซิร์ฟเวอร์ใหม่"

![แท็บสร้างเซิร์ฟเวอร์ใหม่](@assets/CreateNewServer.png)

ใส่ชื่อของเซิร์ฟเวอร์คุณ จากนั้น เลือกซอฟต์แวร์เซิร์ฟเวอร์ โดยมีตัวเลือกดังนี้

- Paper (ค่าเริ่มต้นที่แนะนำ)
- Purpur
- Vanilla
- Fabric
- NeoForge
- Forge
- Velocity

คุณควรที่จะเลือกซอฟต์แวร์ได้หากคุณรู้ว่าคุณทำอะไรอยู่ มิเช่นนั้น ไปที่[หน้านี้](/QSMSharp/th/common/guides/choosing-a-software)
เพื่อเรียนรู้เกี่ยวกับซอฟต์แวร์เซิร์ฟเวอร์

โดยค่าเริ่มต้น เซิร์ฟเวอร์ของคุณจะถูกวางอยู่ที่โฟลเดอร์เก็บเซิร์ฟเวอร์ค่าเริ่มต้นของ QSM
(`C:\Users\username\AppData\Local\QSM\Servers`)
แต่คุณสามารถเปลี่ยนโฟลเดอร์เซิร์ฟเวอร์เป็นที่อื่นได้หากใจคุณปรารถนา

หลังจากนั้น คุณสามารถเลือกเวอร์ชัน Minecraft และเวอร์ชันซอฟต์แวร์เซิร์ฟเวอร์ได้จากรายการสองรายการได้เลย

ท้ายที่สุด กดปุ่ม "สร้างเซิร์ฟเวอร์" ได้เลย! QSM จะทำการดาวน์โหลดซอฟต์แวร์เซิร์ฟเวอร์และนำทางคุณไปยังหน้าจัดการเซิร์ฟเวอร์ใหม่นี้
