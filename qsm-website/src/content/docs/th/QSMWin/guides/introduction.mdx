QSM Windows เป็นซอฟต์แวร์จัดการเซิร์ฟเวอร์ Minecraft สำหรับผู้ที่พึ่งเริ่มต้นในการเปิดเซิร์ฟเวอร์ 
Minecraft ด้วยตนเอง

## การติดตั้ง

<Tabs>
    <TabItem label="ตัวติดตั้ง">
        ตามหาตัวติดตั้ง QSM.Windows เวอร์ชันล่าสุดที่ [GitHub Releases](https://github.com/lines-of-codes/QSMSharp/releases),
        โดยไฟล์ตัวติดตั้งจะมีชื่อคล้าย `QSMWindows-x64-[เวอร์ชัน].exe` จากนั้นคุณสามารถดาวน์โหลดและติดตั้งได้เลย
    </TabItem>
    <TabItem label="ไฟล์ซิป">

## การตั้งค่า Java

คุณจำเป็นต้องตั้งค่า Java สำหรับเซิร์ฟเวอร์ของคุณก่อน โดยคุณสามารถเข้าไปในหน้า การตั้งค่า > จัดการ
Java และนำเข้าการติดตั้ง Java ที่คุณมีอยู่แล้วหรือดาวน์โหลดรันไทม์ Java ใหม่

หากคุณต้องการนำเข้า คุณจะต้องเลือกโฟลเดอร์การติดตั้ง Java ของคุณ 
โดยโฟลเดอร์นั้นอาจอยู่ที่ไหนก็ได้ขึ้นอยู่กับสถานที่ที่คุณเลือกติดตั้ง Java แต่โดยปกติแล้ว 
จะอยู่ภายใต้ `C:\Program Files\<ชื่อผู้แจกจ่าย>\<เวอร์ชัน Java>` เช่น 
`C:\Program Files\Zulu\zulu-21` สำหรับ Azul Zulu 21 หรือ 
`C:\Program Files\Java\jdk-21` สำหรับ Oracle Java SE 21

หากคุณต้องการดาวน์โหลด คุณสามารถไปยังแท็บดาวน์โหลดในหน้าการจัดการ Java เลือกเวอร์ชัน 
Java เลือกบิลด์ล่าสุด และคุณสามารถคลิกดาวน์โหลดได้เลย

หากคุณไม่ทราบว่าจะต้องดาวน์โหลด Java เวอร์ชันใด โดยปกติแล้ว:

- Java 8 สำหรับ Minecraft 1.8 ถึง 1.11
- Java 11 สำหรับ Minecraft 1.12 ถึง 1.16.5
- Java 17 สำหรับ Minecraft 1.17 ถึง 1.19.2
- Java 21 สำหรับ Minecraft 1.20 ถึง 1.21.11
- Java 25 สำหรับ Minecraft 26.1 และใหม่กว่า
