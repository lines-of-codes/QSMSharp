โดยปกติแล้ว QSM Windows จะเก็บข้อมูลแอพลิเคชันไว้ที่ `C:\Users\[username]\AppData\Local\QSM` โดยมีโครงสร้างโฟลเดอร์ดังนี้

<FileTree>
    
- config.json ไฟล์การตั้งค่าแอพลิเคชัน
- Servers/ เก็บเซิร์ฟเวอร์ที่คุณสร้างไว้
- Java/ เก็บการติดตั้ง Java ที่คุณติดตั้งผ่าน QSM
- Logs/ เก็บไฟล์ล็อก QSM
- Backups/ เก็บแบ็คอัพเซิร์ฟเวอร์ ลบแบ็คอัพที่อยู่ในโฟลเดอร์นี้ในกรณีที่มันไม่ลงทะเบียนอยู่ในแอพลิเคชันเท่านั้น
- Downloads/ ปัจจุบันมีหน้าที่เพียงแค่เก็บม็อดแพ็คที่ถูกดาวน์โหลดไว้ คุณสามารถลบเนื้อหาของโฟลเดอร์นี้ได้อย่างปลอดภัย

</FileTree>
