QSM (Quick Server Manager) เป็นซอฟต์แวร์จัดการเซิร์ฟเวอร์สำหรับเกม Minecraft 
โดยในปัจจุบันมีเวอร์ชันแอพลิเคชัน Windows และเวอร์ชันเว็บแอพลิเคชัน

**QSM ไม่รองรับการเปิดเซิร์ฟเวอร์ Minecraft Bedrock Edition** อย่างไรก็ตาม
คุณยังสามารถเปิดเซิร์ฟเวอร์ Minecraft Java Edition แล้วใส่ปลั๊กอินที่ทำให้ผู้เล่น 
Bedrock Edition สามารถเข้าร่วมเกมได้

แม้ว่า QSM จะช่วยทำให้กระบวนการการเปิดเซิร์ฟเวอร์ Minecraft ด้วยตนเองง่ายขึ้น 
คุณยังควรที่จะเข้าใจหลักการพื้นฐานของการเปิดเซิร์ฟเวอร์เกม

Minecraft Wiki ได้กล่าวไว้ดังนี้ (แปลมาจาก[ต้นฉบับ](https://minecraft.wiki/w/Tutorial:Setting_up_a_Java_Edition_server?oldid=3347971#User_knowledge)):

> ในการเปิดเซิร์ฟเวอร์ของคุณอย่างถูกต้อง คุณควรเป็นอย่างยิ่งที่จะมีความรู้อย่างน้อยดังต่อไปนี้
> - ความรู้ทางเทคนิคเกี่ยวกับเกม เช่น คำสั่ง ชังก์ ฯลฯ
> - การใช้ command-line และแก้ไขไฟล์การตั้งค่า
> - ความรู้ด้านเครือข่ายอินเทอร์เน็ตโดยทั่วไป (IP, พอร์ท, ฯลฯ)
> - การตั้งค่าระบบ

ในการเริ่มใช้ QSM กรุณาเลือกระหว่าง QSM Windows และ QSM Web โดยความแตกต่างเป็นหลักคือ QSM Windows จะใช้งานง่ายกว่าสำหรับผู้ที่เริ่มต้นเนื่องจากมันเป็นเหมือนโปรแกรมอื่น ๆ ทั่วไปใน Windows แต่ QSM Web จะสามารถใช้ได้บนทุกแพลตฟอร์ม นอกเหนือจาก Windows
