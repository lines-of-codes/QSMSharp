QSM Web เก็บข้อมูลไว้ที่สถานที่ต่อไปนี้บนระบบปฏิบัติการต่าง ๆ

- Linux: `/var/lib/qsm-web`
- FreeBSD: `/usr/local/etc/qsm-web/`
- Windows: `C:\ProgramData\QSMWeb\`
- macOS: โฟลเดอร์ที่แอพลิเคชันตั้งอยู่

## โครงสร้างโฟลเดอร์

คล้ายธรรมชาติของ QSM Windows, QSM Web มีโครงสร้างระบบไฟล์ดังนี้

<FileTree>
    
    - config.json การตั้งค่าแอพลิเคชัน
    - downloads/
    - logs/
    - qbs/ สถานที่ติดตั้ง QBS
    - servers
        - [server_name]
            - config.qsm.json การตั้งค่าเฉพาะเซิร์ฟเวอร์ของ QSM
            - ...
    
</FileTree>
