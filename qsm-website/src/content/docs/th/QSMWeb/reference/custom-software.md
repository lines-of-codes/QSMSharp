---
title: ซอฟต์แวร์กำหนดเอง
description: รายละเอียดการใช้ซอฟต์แวร์กำหนดเอง
---

QSM Web รองรับการใช้เซิร์ฟเวอร์ซอฟต์แวร์ของตนเอง อย่างไรก็ตาม 
มีข้อจำกัดที่ว่ามันไม่สามารถมีกระบวนการติดตั้งแยกเหมือน Forge หรือ NeoForge ได้

ในการข้ามข้อจำกัดนั้น คุณสามารถรันกระบวนการติดตั้งก่อนและนำเข้าเป็นอาร์ไคฟ์เซิร์ฟเวอร์ทีหลังได้

ฟีเจอร์นี้สามารถ และจะถูกใช้สำหรับ [RCE (Remote Code Execution; การเรียกใช้โคดระยะไกล)](https://en.wikipedia.org/w/index.php?title=Arbitrary_code_execution&oldid=1356261223)
หากไม่ระวัง ดังที่กล่าวในหน้า[ความปลอดภัย](/QSMSharp/th/qsmweb/guides/security/) *อย่า*ให้คนที่คุณไม่เชื่อถือเข้าถึงเซิร์ฟเวอร์ของคุณ
