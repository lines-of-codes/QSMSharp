Simply said, QSM.Web is a program made to run executables for you. QSM.Web is 
not responsible for any malicious code uploaded through the platform.

So you should **NEVER**:

- Create a user for someone you don't trust fully
- Run QSM with administrative privileges (a.k.a run as root)

## Planned Features

QSM.Web plans to implement integration with Firejail or other containerized 
solutions in the future.
