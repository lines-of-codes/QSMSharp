---
title: Custom Software
description: Detailing the use of custom software
---

QSM Web supports using your own server software, however, there is a limitation
that it can't have a separate install process like Forge or NeoForge

To work around that issue, you can run the install process beforehand and 
import it as a server archive.

This feature can, and will be used for [RCE (Remote Code Execution)](https://en.wikipedia.org/w/index.php?title=Arbitrary_code_execution&oldid=1356261223)
if not careful. As noted in [Security](/QSMSharp/qsmweb/guides/security/), 
*Do not* give anyone you don't trust access to your server.
